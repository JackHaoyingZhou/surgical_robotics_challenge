How to use the recorder:

1. The code of the recorder is located in the joint_pos_recorder.py.

2. Feel free to modify it according to your preference (e.g. file name definition). It also includes the method to read the data and there is a read-data sample called read_save_data.py

*** You may need to modified the record_size depends on the length of your recording time. It would be tricky to choose a good recording data size

3. The initial recorder is inserted in the teleoperation scripts to record the joint space data.

4. Using this recorder, you can save your data and be able to repeat your movements (tested).

5. Alternatively, you can rosbag to record.


This tool can be further developed for various usages if you are familiar with python.
